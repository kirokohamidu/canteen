# Uganda Martyrs University Masaka Campus - Online Canteen System

This is a sample PHP/MySQL application for an online canteen management system with three user roles: student, canteen manager, and system administrator.

## Features
- User authentication with role-based access
- Student access to menu items and order placement
- Canteen manager menu item CRUD
- System administrator user management (insert/update/delete users)
- Record retrieval and search support

## Setup
1. Place this folder in your WAMP `www` directory.
2. Create the database and tables using `database.sql`.
3. Update `db.php` if your MySQL credentials differ.
4. Open `http://localhost/pppp/` in your browser.

## Default accounts
- Admin: admin@umu.ac.ug / admin123
- Manager: manager@umu.ac.ug / manager123
- Student: student@umu.ac.ug / student123
